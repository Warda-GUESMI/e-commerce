Application Web E-commerce (Frontend)
Description

Ce projet est une application web frontend développée avec des technologies modernes, permettant la gestion d’un catalogue de produits, d’un panier d’achat et d’un système d’authentification utilisateur.
L’objectif est de proposer une interface fluide, responsive et facilement extensible, prête à être connectée à un backend.

Technologies utilisées
React (avec TypeScript)
Vite
Zustand (gestion d’état)
CSS moderne (responsive design)
Intégration API REST
